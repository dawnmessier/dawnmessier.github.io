Find It Fast

GitHub

A jQuery search autocomplete plugin with built-in WAI-ARIA

Accessibility for the impaired is often overlooked when building a website. It is a lengthy subject to learn and implement. This plugin is designed to cut down on that effort by providing a search autocomplete (form + list) built with necessary WAI-ARIA HTML. Colors are also chosen for those with color-blindness.

For more information on building accessible websites, refer to these sites:

  • Web Accessibility Initiative (WAI)
  • WebAIM

Features

  • Meets WAI-ARIA requirements; built-in form input HTML
  • Handles JSON objects, arrays and AJAX urls
  • Optionally customize callbacks
  • Mobile-friendly

Initialize element with unique ID

Example Code

Include links and position list above input

$('#searchABC').findItFast({
    dataConfig: {
        type: 'json',
        src: countriesHrefJSON,
        maxItems: 7,
        href: 'href',
        timer: 1000
    },
    templates: {
        listItems: {
            position: 'top'
        }
    }
})

Ajax API

$('#searchAPI1').findItFast({
    dataConfig: {
        type: 'url',
        src: '/countries/search?limit=10',
        key: 'name',
        href: 'url',
        timer: 1000
    },
    templates: {
        form: {
            inputName: 'q'
        },
        listItems: {
            includeLinks: true
        }
    }
})

$('#searchAPI2').findItFast({
    dataConfig: {
        type: 'url',
        src: '/countries/search?limit=5&key=code',
        key: 'code'
    },
    templates: {
        form: {
            inputName: 'q'
        }
    },
    ariaConfig: {
        includeLiveRegion: true
    }
})
